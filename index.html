CLF4S

A tiny Scala library to read logs in the Common Log Format, like the following:

127.0.0.1 - frank [10/Oct/2000:13:55:36 -0700] "GET /apache_pb.gif HTTP/1.0" 200 2326

  • Use the LogEntry companion object apply method to parse a line and get Either a Throwable or an actual LogEntry
  • Each LogEntry includes:
    • host: String
    • identity: Option[String]
    • user: Option[String]
    • date: DateTime
    • request: Request
    • status: Int
    • bytes: Option[Long] (missing for 3xx responses)
  • Each Request includes:
    • method: Method (an ADT that models HTTP request methods)
    • resource: String
    • protocol: Protocol (an ADT that models HTTP protocol versions)

Use it in your own project by importing it with SBT:

  • libraryDependencies += "me.baghino" % "clf4s_2.11" % "0.0.1"
